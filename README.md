## ⚙️ Required ENV Variables

| Variable | Description | Required |
|---|---|---|
| `SESSION_ID` | WhatsApp session (base64 or MEGA link) | ✅ Yes |
| `OWNER_NUMBER` | Your WhatsApp number (no +) | ✅ Yes |
| `PREFIX` | Command prefix (default: `.`) | ❌ No |
| `MODE` | public / private / groups / inbox | ❌ No |
| `MONGODB_URI` | MongoDB connection (for premium features) | ❌ No |
| `GEMINI_API_KEY` | Google Gemini AI key | ❌ No |
| `OPENAI_API_KEY` | OpenAI GPT key | ❌ No |
